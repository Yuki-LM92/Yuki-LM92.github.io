趣意書作成支援ツール
試用中
登壇・対談・取材の依頼書（趣意書）作成を支援するWebアプリ。 人物調査→依頼理由整理→文章生成の3ステップで、趣意書のたたきを自動生成する。
課題：依頼書作成の初期工程（人物調査・文章起こし）に時間がかかり、内容の吟味・ブラッシュアップに時間を使えない
解決：コストのかかる初期工程を短縮し、品質向上のための時間を確保
GitHub Supabase Vercel
PR Tools Chrome拡張機能
稼働中
チーム内に散在するHTMLツールをポップアップからワンクリックで起動できるChrome拡張ランチャー。ツールの所在をまとめ、誰でも迷わず使える状態をつくる。
課題：ツールの所在がバラバラ・毎回再配布が必要・Teamsのプレビューでは動かない
解決：拡張機能のポップアップから最新のツール一覧を表示・ワンクリックで起動できる仕組みを構築
Chrome拡張 Manifest V3 JavaScript
SNS分析ダッシュボード
稼働中
Instagram・X・FacebookのCSVデータをブラウザ上で可視化する分析ツール。インストール不要・単体HTMLで動作。
課題：各SNSのデータがバラバラで、毎回手動でグラフを作り直す手間があった
解決：CSVをドラッグ＆ドロップするだけでグラフとKPIが自動生成
HTML / CSS / JS Chart.js CSV処理
プロジェクト管理ツール
試用中
中規模チーム向けのプロジェクト管理ツール。ボードビューとガントビューを1つのHTMLで提供。サーバー不要。
課題：プロジェクトがチャットに分散し、全体像を誰も把握できていない
解決：ボード・ガントで全体を可視化。JSONエクスポートでデータの持ち出し・共有が可能
HTML / CSS / JS SVG localStorage
DXインフォ 自動配信システム
稼働中
DXに関する短いコラムを毎日自動生成・保存する仕組み。Claude APIを活用し、テーマの重複チェック・週別フォルダ整理まで自動化。
課題：毎日手動でネタを考えて書くのが続かない
解決：cronで毎朝7:10に自動生成。3月分30件を一括準備済み
Python Claude API cron Google Drive
AI Daily Digest 自動配信
調整中
毎朝AIニュースをRSSで収集・Discordに自動送信する仕組み。送信済み記事の重複排除機能付き。Claude CLIと連携した日本語要約機能を組み込み中。
課題：AI情報のキャッチアップに時間がかかる・英語記事が多い
解決：毎朝7:20にDiscordへ自動配信。翻訳精度の安定化を継続改善中
Python RSS Discord Webhook cron
