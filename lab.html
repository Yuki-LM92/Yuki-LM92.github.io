ツールの追加方法

新しいツールを作業場に追加する手順

  1. tools/ツール名/ フォルダを作成して index.html を置く
  2. 上の「開発中」セクションのコメントを外して情報を追記する
  3. git add . && git commit -m "add: ツール名" && git push
  4. https://yuki-lm92.github.io/tools/ツール名/ で即確認できる
  5. 完成したら works.html にカードとして追加する
