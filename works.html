趣意書作成支援ツール
試用中
広報マーケティング部門向け。セミナー登壇・対談・取材の依頼書（趣意書）作成を支援するWebアプリ。 人物調査→依頼理由整理→文章生成の3ステップで、趣意書のたたきを自動生成する。
課題：趣意書作成の初期工程（人物調査・文章起こし）に時間がかかり、内容の吟味・ブラッシュアップに時間を使えない
解決：コストのかかる初期工程を短縮し、品質向上のための時間を確保

SNS分析ダッシュボード
稼働中

DXインフォ 自動配信システム
稼働中
社内向けDX小ネタを毎日自動生成してGoogle Driveに保存する仕組み。Claude APIを活用し、テーマの重複チェック・週別フォルダ整理まで自動化。
課題：毎日手動でネタを考えて書くのが続かない
解決：cronで毎朝7:10に自動生成。3月分30件を一括準備済み
Python Claude API cron Google Drive
AI Daily Digest 自動配信
調整中
毎朝AIニュースをRSSで収集・Discordに自動送信する仕組み。送信済み記事の重複排除機能付き。Claude CLIと連携した日本語要約機能を組み込み中。
課題：AI情報のキャッチアップに時間がかかる・英語記事が多い
解決：毎朝7:20にDiscordへ自動配信。翻訳精度の安定化を継続改善中
Python RSS Discord Webhook cron
